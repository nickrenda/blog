Software

Most of my coding projects can be seen on GitHub, but I have decided to start writing about the projects I do on here. For starters, I am going to demonstrate an R package that I just made.

filetree | Easily create simple file trees using R

Install

Assuming you have devtools installed,

devtools::install_github("epwalsh/filetree")

Examples

# Grow a file tree in the current working directory.
tree1 <- grow()

# Draw the file tree in the console.
draw(tree1)
filetree/
   |--R/
   |   |--draw.R
   |   |--get_children.R
   |   |--grow.R
   |   |--utils.R
   |--man/
   |   |--draw.Rd

   |   |--draw.leaf.Rd
   |   |--grow.Rd
   |--DESCRIPTION
   |--LICENSE
   |--NAMESPACE
   |--filetree.Rproj
# Grow a file tree in the current working directory with hidden files.
tree2 <- grow(showhidden = TRUE)
