viShiny | Interactive visualizations

Built using the Shiny framework for R, viShiny is an interactive web application that demonstrates certain statistical concepts. This is an ongoing project of mine and a tool that I use when teaching introductory statistics. viShiny implements a modular design (inspired by gammaram/intRo) that makes adding new topics quick and easy.

To use this app you have two choices. Either navigate to epwalsh.shinyapps.io/viShiny or run the app locally. To run locally, make sure you have R and the R package "shiny" installed, then from within R shiny::runGitHub("epwalsh/viShiny"). Similarly you could clone this repository and cd in the terminal to the root directory ~/.../viShiny. Then start R from within the same terminal window and enter

library(shiny)
runApp()

To report a bug or suggest a feature, submit an issue at github.com/epwalsh/viShiny/issues. Suggestions and contributions are welcome.

filetree | Easily create simple file trees using R

Have you ever wondered how to make a beautiful represenation of all the files within a directory? Not a command-line guru? No problem. Now you can make elegant file trees right from within R thanks to this neat little package complete with themes and the option to show hidden files. You can also export your file tree to a document.

Install

Assuming you have devtools installed,

devtools::install_github("epwalsh/filetree")

Examples

# Grow a file tree in the current working directory.
tree1 <- grow()

# Draw the file tree in the console.
draw(tree1)

filetree as a command line tool

If you use the Linux or Windows operating systems you may be aware that each has its own command line tool to draw a file tree. However, this tool is missing by default from the OS X Terminal. So if you want to use filetree from the command line, I recommend doing the following.

  • Set up a folder called scripts in your home directory. In this folder, make an R script called tree.R. I also have several other R scripts that I use for different themes, my favorite of which is the galactica theme. I call this script tree_galactica.R.
  • Then make a bash script in the same folder called tree.sh.
  • The files and the structure should look like this:
scripts/
   |__ tree.sh
   |__ tree.R
   |__ tree_rebel.R
   |__ ...
    \_ tree_galactica.R
      
# tree.R
library(filetree)
draw(grow())
# tree_galactica.R
library(filetree)
draw(grow(), theme = "galactica")
#!/bin/bash
# tree.sh

tree() {
    if [ ! -z "$1" ]
    then
        if [ "$1" == "galactica" ]
        then
            Rscript ~/scripts/tree_galactica.R
        else
            echo "Unrecognized theme"
        fi
    else
        Rscript ~/scripts/tree.R
    fi

    return 0
}

Then go into the terminal and enter source ~/scripts/tree.sh. After that you can cd to a directory where you want to make a file tree (I do not recommend doing this from your home directory) and enter tree or tree galactica for the galactica theme.
